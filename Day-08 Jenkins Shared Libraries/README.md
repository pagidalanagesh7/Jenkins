## 💡 Why Shared Libraries?

As Jenkins environments grow, maintaining the same pipeline logic across multiple repositories becomes difficult.

Shared Libraries solve this problem by:

- Reusing pipeline code
- Reducing duplication
- Standardizing CI/CD workflows
- Improving maintainability
- Simplifying enterprise pipeline management
