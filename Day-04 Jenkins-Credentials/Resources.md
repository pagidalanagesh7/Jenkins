## Recommended Practice

✔ Create all four credential types in Jenkins.

✔ Authenticate a GitHub repository using SSH Keys.

✔ Log in to Docker Hub using Jenkins Credentials.

✔ Deploy a sample application to Kubernetes using a kubeconfig Secret File.

✔ Build a Jenkins Pipeline without hardcoding any secrets.

---

## Key Takeaway

A secure CI/CD pipeline starts with proper secrets management. Always use **Jenkins Credentials** instead of storing passwords, tokens, or keys directly in your Jenkinsfile or source code.
